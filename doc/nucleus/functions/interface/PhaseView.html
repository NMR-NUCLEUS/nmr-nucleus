PhaseView

PURPOSE

is an extra subGUI to visualize the phase of a T2 signal

SYNOPSIS

function PhaseView(src,~)

DESCRIPTION 

PhaseView is an extra subGUI to visualize the phase of a T2 signal

 Syntax:
       PhaseView

 Inputs:
       src - handle of the calling object

 Outputs:
       none

 Example:
       PhaseView(src,~)

 Other m-files required:
       none
