PURPOSE

is the general import routine for NMR data

SYNOPSIS

DESCRIPTION 

importNMRdata is the general import routine for NMR data

 Syntax:

 Inputs:

 Other m-files required:

       clearAllAxes

       LoadNMRData_driver

       importDataDart
       importDataGeneral
       importDataLIAG

 MAT-files required:
