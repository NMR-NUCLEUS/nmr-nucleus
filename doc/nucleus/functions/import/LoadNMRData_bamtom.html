LoadNMRData_bamtom

PURPOSE

loads BAM TOM data

SYNOPSIS

function out = LoadNMRData_bamtom(in)

DESCRIPTION 

LoadNMRData_bamtom loads BAM TOM data

 Syntax:

 Inputs:

       in.path - data path
       in.fileformat - 'bamtom'

 Outputs:
