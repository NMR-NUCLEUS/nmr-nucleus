LoadNMRData_bgr

PURPOSE

loads standard BGR NMR data

SYNOPSIS

function out = LoadNMRData_bgr(in)

DESCRIPTION 

LoadNMRData_bgr loads standard BGR NMR data 

 Syntax:
       out = LoadNMRData_bgr(in)

 Inputs:
       in - input structure
       in.path - data path
       in.name - file name
       in.fileformat - 'bgr'

 Outputs:
       out - output structure
       out.parData - parameter file data
       out.nmrData - NMR data

 Example:
       out = LoadNMRData_bgr(in)

 Other m-files required:
       fixParameterString

 Subfunctions:
       LoadDataFile
       LoadParameterFile

 MAT-files required:
       none
